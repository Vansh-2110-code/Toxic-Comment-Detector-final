About This Project

Purpose

This application helps moderate online discussions by automatically detecting toxic comments. It's designed to help maintain healthy online communities.

Dataset

Trained on the Toxic Comment Classification Challenge dataset from Kaggle, containing thousands of labeled comments from Wikipedia discussions.

Technology

Uses machine learning (Logistic Regression + TF-IDF) to classify comments across 6 toxicity categories with high accuracy.

Threat

Insult
